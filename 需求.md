# YEZZ 独立站需求梳理 V1

## 1. 项目一句话定位

YEZZ 独立站是一个面向约会、生日派对、朋友聚会和手作体验客户的 **DIY 体验馆预约型官网**。

网站第一阶段的核心目标不是做完整电商，而是通过品牌氛围、项目展示、派对套餐和作品展示，引导客户 **加微信咨询** 或 **提交预约表单**。

---

## 2. 网站类型

第一阶段定位为：

**预约型官网 + 展示型品牌官网**

也就是说，网站需要同时完成两件事：

1. 建立品牌感，让客户觉得 YEZZ 是一个适合来玩、拍照、约会、生日和聚会的手作空间。
2. 引导客户行动，让客户加微信或提交预约需求。

### 第一阶段不做

- 不做完整商城
- 不做购物车
- 不做在线支付
- 不做库存系统
- 不做会员系统
- 不做复杂订单系统

---

## 3. 核心用户人群

网站主要服务以下客户：

### 1. 约会人群

情侣周末约会、纪念日约会、想找轻松互动体验的人。

### 2. 生日派对人群

想办小型生日派对、朋友聚会、儿童生日或包场活动的人。

### 3. 朋友 / 闺蜜聚会人群

想找一个有氛围、能拍照、能一起做东西的活动空间。

### 4. 亲子 / 儿童活动人群

家长带孩子做手工、儿童生日活动、假期活动。

### 5. 上门活动客户

需要 YEZZ 提供上门 DIY 派对、上门手工活动、机构或小型活动支持的客户。

---

## 4. 首页主线

首页主线确定为：

**约会 + 生日派对 + 手作体验**

首页不要把所有产品平均展示，否则会像杂货铺。首页要先让客户感受到场景：

- 周末约会可以来这里
- 生日派对可以在这里办
- 朋友聚会可以一起来 DIY
- 可以做一份独一无二的手作礼物

---

## 5. 核心转化目标

网站最重要的转化动作是：

1. **Add WeChat / 加微信咨询**
2. **Book Now / 提交预约表单**

所有页面都应该围绕这两个动作设计。

常见按钮包括：

- Book Now
- Add WeChat
- Contact Us
- View Projects
- View Party Packages

---

## 6. 网站页面结构

第一版网站包含 6 个核心页面：

1. Home / 首页
2. DIY Projects / 项目页
3. Party Packages / 派对套餐页
4. Gallery / 作品展示页
5. Book Now / 预约页
6. Contact / 联系我们

---

## 7. 页面需求

### 7.1 Home / 首页

**页面目标**：让客户快速理解 YEZZ 是什么，并愿意继续看项目或直接预约。

**首页需要包含**：

1. 顶部 Banner 大图
2. YEZZ 品牌介绍
3. 三大场景入口：约会、生日派对、手作体验
4. 热门 DIY 项目展示
5. 派对套餐入口
6. 客户作品 Gallery 预览
7. 店铺环境 / 氛围图
8. 加微信 / 预约按钮

**首页传达的感觉**：

- 治愈
- 手作
- ins 风
- 适合拍照
- 适合约会
- 适合生日和聚会

---

### 7.2 DIY Projects / 项目页

**页面目标**：让客户知道 YEZZ 有哪些 DIY 项目可以体验。

**展示方式**：首页按场景展示，项目页按品类展示。

**推荐项目分类**：

| 网站展示分类           | 对应现有品类             |
| ---------------------- | ------------------------ |
| Cream Glue DIY         | 奶油胶                   |
| Beads & Accessories    | 串珠、手机链、挂件、车挂 |
| Plaster Painting       | 石膏娃娃                 |
| Fuse Beads / Pixel Art | 拼豆                     |
| Diamond Painting       | 钻石画                   |
| Wooden DIY / ROKR      | 木质 DIY、ROKR、音乐盒   |
| LEGO & Building Sets   | LEGO、积木类             |
| Handmade Gifts         | 饰品、头饰、耳饰、成品   |
| Kids DIY               | 适合小朋友的项目         |
| Premium DIY Kits       | 高客单价 / 复杂项目      |

**注意事项**：

网站不能直接展示内部系统编号，例如：

- 14-木质diy
- 13-磁吸
- 9999-Trash

客户看到的应该是更好理解、更有吸引力的名称，例如：

- Wooden Music Box DIY
- Couple Bracelet DIY
- Plaster Painting Experience
- Cream Glue Phone Case DIY

**项目卡片信息**：

- 项目图片
- 项目名称
- 分类标签
- 价格区间（如 From $35）
- 预计时长
- 适合标签（date / birthday / kids 等）

**项目详情**：第一版不跳独立详情页，点击卡片展开 Modal 显示更多信息，可直接跳转到预约页并预填项目类型。

---

### 7.3 Party Packages / 派对套餐页

**页面目标**：承接高客单价客户，包括生日、聚会、包场和上门活动。

**建议展示套餐**：

#### 1. Couple DIY Date

适合 2 人约会、纪念日、轻松互动体验。

#### 2. Birthday Party Package

适合生日派对、朋友聚会、小型包场。

#### 3. Kids DIY Party

适合儿童生日、亲子活动、学校假期活动。

#### 4. Mobile DIY Party

适合上门派对、上门手作活动、机构活动。

**价格策略**：

第一版价格可以不写死，可使用：

- From $xx per person
- Contact us for a custom quote
- Price varies depending on group size and project selection

这样后期比较好调整。

**套餐卡片信息**：

- 套餐名称 + 描述
- 包含项目列表
- 适用人数
- 价格指示
- CTA 按钮 "Book This Package"

---

### 7.4 Gallery / 作品展示页

**页面目标**：通过真实图片种草客户，建立信任感和氛围感。

**Gallery 分类建议**：

- Couple DIY
- Birthday Party
- Kids DIY
- Handmade Gifts
- Store Vibes
- Customer Works

**展示重点**：

这个页面不需要大量文字，重点是图片好看、有氛围、有真实感。图片点击可放大查看。

---

### 7.5 Book Now / 预约页

**页面目标**：收集客户预约线索，方便人工跟进。

**表单字段**：

- Name / 姓名
- Phone / 手机号
- WeChat / 微信号
- Email / 邮箱
- Preferred Date / 意向日期
- Number of People / 人数
- Activity Type / 活动类型
  - Date
  - Birthday
  - Friends Gathering
  - Kids DIY
  - Mobile Party
- Interested DIY Project / 感兴趣的项目
- Message / Special Request / 备注或特殊需求

**提交后提示**：

英文：

> Thanks for your enquiry. Our team will contact you shortly to confirm availability and details.

中文：

> 感谢提交预约，我们会尽快联系你确认时间、人数和活动细节。

**表单提交后**：

- 数据存入后台，方便查看管理
- 自动发邮件通知店主
- 如果用户留了邮箱，也发一封确认邮件给用户

---

### 7.6 Contact / 联系我们

**页面目标**：让客户能找到 YEZZ，并建立基础信任。

**页面内容**：

- 店名 YEZZ
- 地址
- 营业时间
- 微信二维码
- Instagram
- 小红书
- 电话
- Email
- Google Map
- 预约说明
- 到店提醒

---

## 8. 产品展示逻辑

产品展示采用两套逻辑：

### 8.1 首页按场景展示

首页不直接堆产品，而是按客户需求展示：

- Date Ideas / 约会好去处
- Birthday Party / 生日派对
- DIY Experience / 手作体验
- Kids Activity / 亲子活动
- Friends Gathering / 朋友聚会

### 8.2 项目页按品类展示

当客户进入项目页后，再按具体项目分类展示：

- 奶油胶
- 串珠 / 挂件
- 石膏娃娃
- 拼豆
- 钻石画
- 木质 DIY
- LEGO / ROKR
- 饰品 / 手作礼物
- 儿童 DIY

## 9. 核心用户路径

### 路径 1：约会用户

首页看到 Date Ideas → 进入 Couple DIY Date → 查看项目和作品 → 点击 Book Now 或 Add WeChat

### 路径 2：生日派对用户

首页看到 Birthday Party → 进入 Party Packages → 查看套餐和 Gallery → 提交预约表单

### 路径 3：普通 DIY 用户

首页看到 DIY Projects → 选择项目分类 → 查看项目介绍 → 加微信确认时间和价格

### 路径 4：上门活动用户

首页或派对页看到 Mobile DIY Party → 了解上门活动 → 提交需求 → 人工报价

### 前台功能

- 首页展示
- 项目分类展示
- 项目卡片 / 项目详情展示（Modal 形式）
- 派对套餐展示

### 后台功能（CMS）

- DIY 项目管理
- 项目分类管理
- 派对套餐管理
- Gallery 图片管理
- 预约表单提交记录查看
- 网站全局设置（联系方式、微信二维码、SEO）

---

## 11. 第一版暂不做的功能

以下功能第一阶段不建议做：

- 购物车
- 在线支付
- 库存管理
- 会员系统
- 优惠券
- 复杂订单系统
- 自动排班
- 多门店系统
- 用户评论系统
- 积分系统
- 在线预约日历（选时间段）
- 项目独立详情页（用 Modal 代替）
- 博客 / 教程内容
- 高级动画（视差滚动、WebGL）
- 数据分析 / 埋点

原因：这些功能会增加开发复杂度，但不一定能马上提升转化。第一阶段最重要的是验证客户是否愿意通过网站咨询和预约。

## 12. MVP 范围

第一版 MVP 建议做成：

**标准官网 + 预约表单 + CMS 内容管理**

### MVP 页面

1. Home
2. DIY Projects
3. Party Packages
4. Gallery
5. Book Now
6. Contact

### MVP 核心按钮

- Book Now
- Add WeChat
- View Projects
- View Party Packages

### MVP 目标

让客户在 1 分钟内明白：

1. YEZZ 是什么
2. 适合什么场景
3. 有哪些 DIY 可以做
4. 能不能办生日 / 派对
5. 怎么预约

## 13. 设计风格方向

用户选择的风格为：

**手作、治愈、ins 风**

视觉关键词：

- 温暖
- 干净
- 治愈
- 有手作感
- 适合拍照
- 不要太像低价杂货铺
- 不要太复杂
- 不要太商业化

### 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| **背景主色** | 暖奶油白 | `#FDF6F0` |
| **品牌主色** | 柔雾粉 | `#E8A0BF` |
| **辅助色** | 鼠尾草绿 | `#9CAF88` |
| **文字主色** | 暖炭灰 | `#3D3D3D` |
| **文字次要** | 暖灰 | `#8A8A8A` |
| **强调/按钮** | 焦糖棕 | `#C1785C` |
| **点缀** | 淡薰衣草 | `#D8C3E3` |

## 15. 当前需求结论

YEZZ 独立站第一版应建设为一个 **预约转化型 DIY Studio 官网**。

网站围绕"约会 + 生日派对 + 手作体验"展开，通过首页氛围、项目展示、派对套餐、作品 Gallery 和预约表单，引导客户加微信或提交预约。

第一阶段不做完整电商，而是优先完成品牌展示、项目展示和预约转化。后续再根据业务情况扩展在线支付、购物车、库存和会员系统。

**技术方案概述**：Next.js 15 + TypeScript + Tailwind CSS + Sanity CMS + next-intl + Resend + Vercel
